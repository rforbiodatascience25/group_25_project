##  Original Data: Pancreatic Adenocarcinoma TCGA PanCancer data

https://www.cbioportal.org/study/summary?id=paad_tcga_pan_can_atlas_2018

PAAD TCGA Pan-Cancer Atlas (2018) contains pancreatic adenocarcinoma patient data including demographics (age, sex, race, ancestry), genomic instability metrics (mutation count, TMB, fraction genome altered), biological scores (aneuploidy, hypoxia, MSI), cancer staging/clinical codes, and survival outcomes with therapy history.​
